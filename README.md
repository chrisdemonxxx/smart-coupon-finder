# Smart Coupon Finder — Free Automatic Coupon Codes

Free alternative to Honey and Capital One Shopping. Automatically finds and applies coupon codes at checkout.

## Features
- 🔍 Auto-detects coupon fields on checkout pages
- 💰 Tests thousands of known discount codes
- ⚡ Works on all major stores (Shopify, WooCommerce, Magento, etc.)
- 🆓 100% free, no account needed
